# Software Development Experiment

## 1. 项目简介

本仓库为 XJTU CS Software Development 2026 课程实验合集，共包含四个实验目录：

- `exp1`：Rainfall Alert System
- `exp2`：SCS-CN Runoff Model
- `exp3`：Reservoir Optimization
- `exp4`：Flood Inundation Analysis

本项目的实验开发过程使用 Codex 配合 GPT-5.5 完成，主要包括实验需求理解、代码实现、测试验证、结果可视化、Prompt Log 记录和 README 文档整理。每个实验目录均保持独立并且各自包含README，便于查看、运行和复现实验结果。
